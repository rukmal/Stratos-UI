# Apache Stratos UI

## Usage

### Cleaning files up

After cloning this repo, clean up existing html files by running:
```bash
$ sh cleanup.sh
```

### Compiling from Jade

To compile the site from jade, run:
```bash
$ sh compile.sh
```

### Recompiling from Jade

To recompile the site (i.e. remove old files and compile), run:
```bash
$ sh recompile.sh
```
